# VS Code 확장 ID 목록 (Cursor에서 똑같이 설치용)
# 사용법:
# 1. VS Code에서 터미널 열고 실행: code --list-extensions
# 2. 출력된 목록을 복사해서 아래에 한 줄에 하나씩 붙여넣기 (이 주석 줄은 지워도 됨)
# 3. 프로젝트 루트에서 실행: .\scripts\sync-extensions-to-workspace.ps1
# 4. Cursor에서 이 폴더 열고 "워크스페이스 권장 확장 설치" 클릭
#
# 예시 (아래는 이 프로젝트에 자주 쓰는 확장 예시):
dbaeumer.vscode-eslint
bradlc.vscode-tailwindcss
esbenp.prettier-vscode
